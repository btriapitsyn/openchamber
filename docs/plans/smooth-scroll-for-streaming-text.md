# Smooth Scroll for Streaming Text Animation

## Intent

Зробити плавний скрол вниз коли з'являються нові рядки тексту асистента під час анімації. Замість миттєвого стрибка на нову позицію, скрол повинен плавно рухатись вниз разом із появою нового контенту.

## Що потрібно врахувати

### Частота оновлень
- Нові рядки з'являються кожні 100ms (`StreamingAnimatedText.tsx:90`)
- Це означає 10 оновлень на секунду
- Автоскрол викликається при кожній зміні контенту (`handleMessageContentChange`)

### Поточна реалізація
- `useChatScrollManager.ts` використовує миттєвий скрол
- `scrollRef.current.scrollTop = scrollRef.current.scrollHeight`
- Викликається при кожному рості контенту

### Бажаний результат
- Плавний рух скролу вниз коли з'являються нові рядки
- Візуальний ефект "хвилі" - текст анімується і скрол плавно слідує за ним
- Відсутність ривків або переривань анімації

### Технічні обмеження
- Анімація тексту: `blurAndSharpen`, 250ms duration
- Інтервал між рядками: 100ms
- Автоскрол повинен працювати тільки коли `shouldAutoScroll === true`
- Не повинен заважати користувачу який проскролив вгору
